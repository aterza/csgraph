    Renderer

    this class is responsible for the csound score reading It is expected to work well with any csound file, whether it'd be a '.sco', a '.csd' or an '.orc' file (for which it should do nothing of course) and a .csg file (the CsGraph description language)

def render
  clear unless self.score_lines.empty?
  read
                    self.score_lines.each do

                            sl.render(self.output_stream)
                    end
