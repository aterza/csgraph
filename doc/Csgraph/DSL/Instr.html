    class Csgraph::DSL::Instr

    Definition

    Technically, Definition, is the class that holds the full definition of a graph to build for a given instrument (or group of instruments)

    An example of a very basic definition is

    CsGraph.define do

  instr 1,2,3 do

   line p2, p2+p3, p5, p5, :thickness => p4

  end

end

    which means “draw lines taking pfield 5 as y start point, pfield 5 as y end point, and adjusting the thickness of the line to pfield 4”
